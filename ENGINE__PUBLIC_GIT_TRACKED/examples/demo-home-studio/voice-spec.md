# Voice Spec — Sweet Fern Sugaring Studio (demo persona)

A `voice_spec` reference for the studio persona.

## Register
Warm, plain-spoken, reassuring. Talk to a nervous first-timer like a knowledgeable
friend, not a brand. Short sentences. No jargon, no hype, no shame. Concrete and
practical — tell people exactly what to do and why. A light, kind sense of humor is
welcome. An emoji now and then is fine in a social post.

## Hard rules
- No contractions rule either way — sound like a real person.
- No corporate wellness clichés ("self-care journey", "pamper yourself").
- Never overpromise (no "permanent", no "painless"). Say "gentler", "calmer", "lasts".

## Truth
Only claim what the studio actually does. No medical claims.
