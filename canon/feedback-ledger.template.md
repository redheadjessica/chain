# Feedback Ledger (template)

Your reactions to drafts, accumulated into durable rules. Agents read this before
drafting; **newest entries override every spec** where they conflict. It earns its
value the first time you react to a draft — create it empty and let it grow.

> Replace this placeholder header with your first entry when it happens.

Entry format:

```
## YYYY-MM-DD — short title
**Context:** what was being written
**Feedback:** what you actually said or changed
**Rule:** the durable lesson, stated for an agent
```
