# Format: long_post

A longer social post that develops a point more fully than a short_post but is still
a post, not an article. Default channel: LinkedIn.

## Mechanical (linted)
- Roughly 250–600 words.
- Fold-aware opening on LinkedIn.
- Paragraph breaks / whitespace for readability.

## Judgment
- One thesis with two or three supporting beats; still one idea, more room.
- If it wants headings and sections, it probably wants to be an `article` instead.
